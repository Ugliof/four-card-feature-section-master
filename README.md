# Four Card Feature Section

Projeto desenvolvido para praticar habilidades de **desenvolvimento Front-End**, recriando um layout moderno utilizando HTML e CSS.

## Tecnologias utilizadas

* HTML5
* CSS3
* CSS Grid
* Flexbox
* Media Queries

## Objetivo

Praticar a construção de layouts responsivos e organização de código utilizando boas práticas de desenvolvimento Front-End.

Durante o projeto foram aplicados conceitos como:

* Estrutura semântica em HTML
* Layout com CSS Grid
* Organização de estilos com CSS
* Responsividade para diferentes tamanhos de tela

```
.
├── index.html
└── style.css
```
